# Syntaxe et sémantiques

Dans cette section, nous étudierons une à une les différentes fonctionnalités
de Rust, chacune dans son propre chapitre. Si vous souhaitez commencer votre
apprentissage de Rust à-partir des bases, cela me semble être un bon endroit
pour commencer.

Ces sections forment aussi, en quelque sorte, un référence sur chaque concept.
Ainsi, si vous lisez un autre tutoriel et êtes confus par un concept présenté,
vous devriez trouver une explication quelque part ici.
